# Wyszukiwarka_na_SW

Dane do projektu zostały pobrane ze strony kaggle.com a później załadowane do SQLite (5 pkt za źródło oraz 10 pkt za SQLite).
https://www.kaggle.com/datasets/kashifsahil/16000-movies-1910-2024-metacritic

Zbiór zawiera około 15 tysięcy obserwacji (8 pkt).

Apliakcja działa za pomocą Flaska (15 pkt).

2 dodatkowe cechy w zbiorze danych (1 pkt).

5 filtrów wyszukiwania (5 pkt).

Zastosowano TF-IDF (10 pkt).

Do podobieństwa termów zastosowano odległość Levenshteina (5 pkt) oraz miary podobieństwa Jaccarda, cosinusa oraz LSI (20 pkt).

5 typów wykresów (KPI, wykres kolumnowy, wykres kołowy, tabela oraz chmura słów (20 pkt).

Łącznie punktów: 99
